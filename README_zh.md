## 简介

这是一个允许玩家使用任意组合制作料理的 Minecraft 模组。

### 核心特性

- **自由烹饪系统** - 使用任意食材组合制作料理
- **配方解锁机制** - 锅料理配方默认锁定（可通过配置取消），只有制作过一次才能解锁
- **探索式玩法** - 通过提示系统引导玩家尝试不同组合，获得"炼金"式的烹饪体验
- **动态属性** - 根据原料动态计算料理的饱食度和饱和度
- **配方记录** - 可以将成功的组合记录到配方书中

## 系统详解

### 无名料理系统

1. **动态属性计算**
   - 根据原料的饱食度和饱和度加权计算
   - 考虑原料的营养价值和稀有度
   - 支持配置权重和计算方式

2. **原料追踪**
   - 制作时记录所有原料信息
   - 鼠标悬停时显示原料列表
   - 支持NBT敏感原料的识别

3. **命名与配方化**
   - 使用工作台或特殊工作台为无名料理命名
   - 命名后可创建配方书
   - 配方书可存储到玩家个人配方集合中

### 解锁机制

- 初始状态下所有厨锅配方隐藏
- 制作成功后解锁对应配方
- 配方可通过配方书记录和分享
- 支持配置文件调整解锁难度

## 配置选项

```
# 通用设置
general {
  # 是否隐藏原始配方
  hideOriginalRecipes=true
  
  # 解锁难度：easy(制作1次)、normal(制作3次)、hard(制作5次)
  unlockDifficulty="normal"
  
  # 是否启用提示系统
  enableHints=true
}

# 属性计算
calculation {
  # 饱食度计算权重
  hungerWeight=1.0
  
  # 饱和度计算权重  
  saturationWeight=1.0
  
  # 是否考虑原料稀有度
  considerRarity=true
}
```
