# 项目概述

## 项目介绍

**Nameless Dishes (无名料理)** 是一个 Minecraft Forge 模组，为 Minecraft 1.20.1 版本提供了一个"自由烹饪"系统。该模组允许玩家在不使用预设配方的情况下，使用任意食材组合来制作料理，提供了一种类似"炼金术"的烹饪体验。

## 核心特性

### 自由烹饪系统
- 玩家可以使用任意食材组合进行烹饪
- 不需要预设配方，通过实验发现新的料理组合
- 料理的营养价值会根据食材自动计算

### 动态食物属性
- 根据食材自动计算饥饿值和饱和度
- 食材信息保存在物品 NBT 数据中
- 鼠标悬停显示料理包含的食材列表

### 模组兼容性
- 完全支持 Farmer's Delight (农夫乐事) 的烹饪锅
- 可扩展的兼容层架构，便于添加其他烹饪模组支持
- 配置驱动的系统，可启用/禁用特定功能

## 技术信息

- **目标平台**: Minecraft 1.20.1 + Forge 47.4.0
- **开发语言**: Java 17
- **开源许可**: MIT License
